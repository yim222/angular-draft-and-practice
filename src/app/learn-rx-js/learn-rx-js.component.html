pipe()

pipe is a method on Observable which is used for composing operators.
See more here

The pipe on Rxjs is for give U pipe functions, to each other so each function will do something, and pass the value returned to the next function with operator. His words : “In RxJS, the idea is that you create a pipeline of operators (such as map and filter) that you want to apply to each value emitted by a source observable, of(1,2,3) in this example. This approach lets you create small, reusable operators like map and filter, and compose them together when needed using pipe. “

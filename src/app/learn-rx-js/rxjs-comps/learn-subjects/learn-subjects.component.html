There are also a few specializations of the Subject type: BehaviorSubject, ReplaySubject, and AsyncSubject.

Behavior Subject

Represent the latest values emitted.

BehaviorSubjects are useful for representing "values over time". For instance, an event stream of birthdays is a Subject, but the stream of a person's age would be a BehaviorSubject.

Behavior said - some value that have a behave and change over time. Like age. The date of birth, unlike that - is permanent and doesn't changed over time.

Replay Subject

Replay subject - it's like the behavior that return the last, but you can also define to it how much old values to return.
See the function example, that when the other subscriber subscribe it's return some of the old values.

Async Subject

Async subject - it's happen only when complete and only last
See the function example, that when the other subscriber subscribe it's return some of the old values.

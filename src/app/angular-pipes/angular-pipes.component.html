Angular-Pipes

The pipes of angular (Not the same as the pipes of rxjs - I think) is like good tools , for generate in the display easily things and values, like upper cases, or currency symbol, converting to percentages... and more

Basic

Very simple it can be written like that inside curely brackets (like angular values inside templates) :
{{'{{' + 'myvalue | mypipe}' + '}'}}
If U need to provide params (which available in some pipes) U provide them with :'mypar'. See the docs...

Pipe idea

In general pipes is prepared functionality of angular to convert(transform) values, like date and currency in quick and simple way.
pipe operator ( | )

Example

Date without pipe : {{currentDate}}

Date WITH pipe : {{currentDate | date}} - see how the pipe cuts the trash

Pipe with params (each of them separated by ':' : {{currentDate | date: 'shortDate'}} - The param change it to short-Date

Pipe with parmater as component property: {{currentDate | date: datePar1}} - The param here is component property

Pipe with Using my locale : {{currentDate | date: 'shortDate': '': 'lin-he'}} - It should be d/m/y like in Israell

Not linked to pipes only but remember that U can use in typescript get - to format a variable (and use the value) X = {{x}}

Pipe with parmater as component property: {{currentDate | date: datePar2}} - The param here is component property

  • Decimal - for display decimal number like U set. Or by locale (number)

    Have two numbers.
    one with many decimal digits = {{manydigits}},
    second with no decimal = {{noDigits}}
    third only decimal values {{decimal}}
    U can set the format min.min-max. (it will always show integers)
    So for example that : 1.2-5 Will show on each :
    Many digits = {{manydigits | number: dec1}},
    no decimal = {{noDigits | number: dec1}}
    only decimal values {{decimal | number: dec1}}
    This {{dec2}} will show on each :
    Many digits = {{manydigits | number: dec2}},
    no decimal = {{noDigits | number: dec2}}
    only decimal values {{decimal | number: dec2}}
    Locale {{myLocale}} will show on each :
    This {{dec2}} will show on each : Many digits = {{manydigits | number: undefined :myLocale}},
    no decimal = {{noDigits | number: undefined :myLocale}}
    only decimal values {{decimal | number: undefined :myLocale}}

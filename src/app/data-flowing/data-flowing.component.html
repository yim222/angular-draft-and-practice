Steps to create outputs (event-emitters, data-flow)

- create eventEmitter inside the child - with @Output, that will emit the values.

-define function in the parent that get it.

- Then U can use the the Output as event inside your component.

Pass the parent function as event to the child directive with round brackets. Like this - (parentConnections2)

Notice - the output is inside the component that should out it (the child)

Don't forget the annotation (decorator) @Output
